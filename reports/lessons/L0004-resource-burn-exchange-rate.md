[Codex 2026-06-22] 判斷「是不是燒太兇」不能只數自己花了幾張，要看它把對手推到哪裡。

本場 Set 5 的高風險接球線花了兩張事件與 3 Guts，但京谷 OP7 同時逼對手棄 1、再投入 3 張攔網，共換掉四個身體；後續及川支付 3 Guts 回收岩泉，再用最後一張手牌把 OP 推到 7，對手只剩兩個身體而無法完成 receive → toss → attack。這些花費都有直接的不可逆回報。

反例是 Set 1 把唯一 R5／B3 花在沒有得分或防守收益的地方。通則：**花資源前先說出要換到什麼；答不出對手將失去哪張牌、哪個身體或哪條合法線，就先保留。**
