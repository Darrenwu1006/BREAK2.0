**堆墓的來源是「棄牌成本」，不是某張卡的 dump 效果**（這條一開始寫錯過，已釘正）：
- **大見（P02-085）／小作（P02-035）＝廉價 miller，無 Guts 成本**：大見成本＝棄 1 張稲荷崎手牌抽 2；小作成本＝把自己（WS 稲荷崎）從手牌棄掉。棄掉的稲荷崎即 +1 墓，還附帶抽牌/RCV 加值。**這兩張才是 Set 1 就能點火的卡**——一手同時堆墓＋防守＋（大見）挖牌。各 ×3，Set 1 摸到是合理期望值。
- **北信介（P02-024）／宮侑（P02-017）＝payoff，不是 miller**：北信介＝棄 1＋付 3 Guts → 接球+1、從 **Event 區**回收稲荷崎；宮侑＝付 3 Guts 且 **gated 在棄牌區已有 6 種**才能撿 WS/MB 回手。**「等 Guts 夠才下北信介」是對的**，別無腦拍當燃料——北信介是 WS、宮侑撿得回，但要 6 種先成才撿得回（雞生蛋），拿來加速點火不成立。

所以「讓 Set 1 來堆墓」要修正：Set 1 不會自動堆墓，它的真實任務是 ① survive 對手早期 tempo、②用大見抽濾**挖出 miller/引擎**、③湊 Guts 準備。**讓大見/小作早一個 Set 上線，6 種堆墓才有機會在 Set 2 完成。**

**贏線校正**：稲荷崎常被自我認知成「堆墓爆發」，但實戰多是「厚 DP（壓對手 OP）＋回收續航磨死」。6 種是保險不是唯一解——不必等 6/6 才開始贏。

**配套心法（對後期高火力對手如白鳥沢）**：DP 留到對手 OP 峰值後段；早期能擋就擋、擋不住認賠保牌，**絕不把好防禦牌投進必敗判定**（反例＝太早打掉防禦牌→接不住連續高攻而敗）。出牌順序優先「雙用途」卡入位（大見/小作→receive 成本即堆墓、角名→attack），北信介/宮侑等資源到位再下。對照技能時機的資源紀律見 [[L0001-iwaizumi-skill-timing]]。
