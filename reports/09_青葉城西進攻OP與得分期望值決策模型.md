> **撰寫日期**：2026-06-20 ｜ **版本**：v1.0 ｜ **分析者**：[Gemini 2026-06-20]
> **核心目標**：撇開複雜技能的干擾，純粹從「數值對決」與「手牌代價」的數學期望值出發，指導玩家在發球/攻擊階段，如何精確計算是否需要使用及川、岩泉技能或事件卡推高 OP。

## 1. 得分期望值數學公式與判準

在 BREAK TCG 中，得分是由一次簡單的不等式判定決定的：
$$\text{得分機率 } P(\text{Score}) = P(\text{對手 DP} < \text{我方 OP})$$

我們每次決定是否「使用技能（棄手牌）」或「打出事件卡」使數值 $+1$ 或 $+2$ 時，必須衡量：
$$\Delta P = P(\text{DP} < \text{OP} + \Delta) - P(\text{DP} < \text{OP})$$
如果 $\Delta P$（得分率的提升）無法彌補您付出的手牌代價，那麼這次加點就是**負期望值**的，應予拒絕。

## 3. 「及川徹發球」得分期望值與技能決策

及川徹 (HV-P01-033) 的基礎發球為 5。開技能（棄 1 手牌）或打事件卡可使其變為 6。

### 📊 發球 OP=5 vs OP=6 的得分期望值對比：

#### 1. 對 音駒：
* **OP=5 時**：對手手牌有海信行 (RCV5) 即可直接接下（DP=5）。音駒接下概率約 **60%**，我方得分率 **40%**。
* **OP=6 時**：音駒要接下，必須登場夜久衛輔，且**被迫棄 1 手牌**使 RCV 變 7（DP=7）。若音駒不棄牌或無夜久，則直接漏球。我方得分率提升至 **70%**（且能強制削弱音駒手牌）。
* **決策**：**期望值極高**。只要我方手牌 $\ge 5$，建議開技能或打事件卡到 **OP=6**。

#### 2. 對 白鳥沢：
* **OP=5 時**：白鳥沢有大平獅音 (RCV5)，且山形 (RCV5) 也能接。接下概率約 **55%**。
* **OP=6 時**：白鳥沢只有山形滿足「我方手牌 $\le 4$」時能達到 RCV6。其他時候白鳥沢無卡可單獨接下 OP6。我方得分率提升至 **80%**。
* **決策**：**期望值極高**，建議推到 **OP=6**。

#### 3. 對 稻荷崎：
* **OP=5 時**：稻荷崎有赤木路成 (RCV5) 或其他角色接球，接下概率約 **40%**。
* **OP=6 時**：稻荷崎只有上手白板赤木路成 (P02-034 RCV6) 才能接下（D03-011 的 RCV+2 技能開局首回合無 Guts 無法發動）。我方發球得分率提升至 **80% 以上**。
* **決策**：**期望值極高**。首回合發球建議推到 **OP=6**，對手在首回合高機率漏球。

#### 4. 對 梟谷：
* **OP=5 時**：梟谷能以小見春樹 (RCV5) 接下，概率約 **50%**。
* **OP=6 時**：梟谷需上手白板小見春樹 (P01-050 RCV6) 或小見春樹 (HV-P01-047 RCV5，在Guts奇數時+1) 才能接球。我方得分率提升至 **75%**。
* **決策**：**期望值高**，建議推到 **OP=6**。

## 4. 「進攻攻擊（Attack）」OP 黃金臨界點

在托球與攻擊階段，我方進攻會面臨對手的**攔網（Block）**。四大對手的攔網 DP 極限大部分落在 **6 點**。

### 🎯 為什麼 OP = 7 是黃金臨界點？
* **音駒**：黑尾 (BLK3) + 犬岡 (BLK3) = DP 6。
* **白鳥沢**：天童 (BLK3 + 投幣正面 3) = DP 6。
* **稻荷崎**：大耳 (BLK3) + 銀島 (BLK3) = DP 6。
* **梟谷**：尾長 (BLK3) + 鷲尾 (BLK3) = DP 6。

這意味著，如果您的進攻 **OP = 6**，對手只要登場雙攔網（或天童投幣正面），就能以 **DP=6** 接下。您的進攻會被高機率防住。
但只要您的進攻 **OP = 7**：
1. 對手雙攔網（DP=6）判定失敗，直接得分。
2. 對手若要防住，必須登場 3 張攔網卡（這在實戰中對手極難湊齊同名以外的 3 張 BLK 卡，且會耗盡其手牌），或者我方已經有岩泉一的登場限制。
3. **結論：OP=7 是強攻的「絕對斬殺線」。**

## 5. 「代價 vs 收益」手牌期望值決策清單

當您在糾結要不要開及川/岩泉技能或事件卡時，請默念這套決策流程：

```
                    【我方準備進行發球或攻擊】
                                │
                 ┌──────────────┴──────────────┐
             我是發球？                     我是攻擊？
                 │                             │
         ┌───────┴───────┐             ┌───────┴───────┐
   對手手牌 ≥ 7?   對手手牌 ≤ 6?   目前 OP 為 5-6?   目前 OP 已達 7?
         │               │             │               │
         ▼               ▼             ▼               ▼
     【不開技能】    【開至 OP=6】   【開技能/事件】   【拒絕效果】
   對手手牌太多，   對手難以接球，   推上 OP=7 斬殺線  OP已達臨界點，
   1換1無威脅，    高機率直接得分。  穿透對手雙攔網。  再棄手牌是浪費。
   我方應保留手牌。
```

### 💡 實戰名言：
> **「多餘的數值是毒藥。」**  
> 如果 OP 已經是 7，再棄 1 手牌開岩泉技能把 OP 推到 9，得分機率依舊是 100%，但您卻永久失去了一張能在下回合防守登場的角色卡。這在 TCG 中被稱為「無效溢出」。
