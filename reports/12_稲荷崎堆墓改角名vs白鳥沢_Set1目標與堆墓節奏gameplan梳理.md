## 1. 核心結論

1. **「讓 Set 1 來堆墓」是機制誤會**：Set 1 棄牌區＝0、6 種＝0/6，Set 1 一張都沒堆到墓。堆墓實際在 Set 2 Turn 4，靠大見/小作等技能的**棄牌成本**才點火。
2. **6 種堆墓沒在 Set 2 前完成（Set 2 結束僅 3/6，Set 3 Turn 3 才達成），你卻 Set 2 就贏了** → 贏 Set 2 不需要完成 6 種。真正贏線＝厚 DP（平均 6.14 壓 AI OP 3.5）＋部分引擎回收續航，堆墓 6/6 是保險不是唯一解。
3. **Set 1 更積極的目標＝讓廉價 miller 上線**：大見/小作**無 Guts 成本**，Set 1 就能用「棄稲荷崎成本」一手做到堆墓＋RCV 防守＋（大見）抽 2 挖牌。提早上線 → 6 種有機會如你所願在 Set 2 完成。北信介/宮侑是 Guts／6 種 gated 的 payoff，照原本留著等資源即可。
4. **堆墓 vs 防禦牌的心法本場做對了**：絕不把好防禦牌投進必敗判定；早期能擋就擋、擋不住認賠保牌。

## 3. Set 1 逐回合還原（為什麼沒堆墓、為什麼丟 Set）

玩家全 Set 1 都是防守方（T2/T4/T6 皆 defense-choice receive），白鳥沢握發球主動：

| 回合 | 防守判定 | 反打 | 結果 |

| T2 | 赤木路成 receive，DP6 vs OP4 → **成功** | 銀島結 attack OP2，被五色工攔下 | 守住 |
| T4 | 銀島結 receive，DP4 vs OP0 → **成功**（宮侑 Guts 3 準備此回達成）| 宮治 attack OP3，被天童覚攔下 | 守住 |

- **防守判定 T2/T4 都贏**，沒有「被連續高攻打穿」。Set 1 的失，是 T6 主動不登場認賠。
- T6 牌況（step 30）：手牌＝小作裕渡[3/0/0/0/0]、事件、宮治[1/3/0/2/2]。兩張角色 receive 值都是 0，丟進 receive＝ DP 0 必敗還賠牌 → 認賠保牌是正確止損。
- **運氣註腳**：defense-choice（選 receive）發生在抽牌**之前**；宮治（block 3，可單卡攔網）是選完 receive 後才抽到，已無法改走攔網。屬抽序卡點，非誤判。
- **整 Set 棄牌＝0**：引擎卡（北信介/大見/小作）Set 1 都沒進場做事 → 堆墓自然是 0。

## 4. 三個問題的正解

### Q1：「讓 Set 1、Set 2 前完成堆墓」合理嗎？
方向合理（稲荷崎是讓 X 追三的慢牌），但**因果要修**：Set 1 不產墓。Set 1 的真實價值＝① survive 白鳥沢早期 tempo、②**挖引擎卡**、③湊 Guts 準備。把目標從「Set 1 堆墓」改成「**Set 1 把回收引擎挖出來並上線**」。

### Q2：Set 1 有沒有更積極的目標？有——但靠大見/小作，不是北信介。
**讓大見（P02-085）/小作（P02-035）在 Set 1 上線。** 無 Guts 成本、從手牌打：成本＝棄 1 張稲荷崎（即 +1 墓），效果＝稲荷崎 RCV +1/+2（防守）＋大見抽 2（挖牌）。一手同時堆墓＋防守＋挖。你有大見×3、小作×3，Set 1 摸到是合理期望值，draw step 別 pass、拿來找它們。
**北信介（3 Guts、回收 Event 區）與宮侑（3 Guts、gated 在 6 種已成才能撿 WS/MB）是 payoff，不是 Set 1 點火工具**——「等 Guts 夠才下北信介」的直覺是對的，別把它當燃料亂拍（6 種沒完成前宮侑也撿不回它，雞生蛋）。

### Q3：堆墓 vs 防禦牌的心法與出牌順序
- **心法**：對白鳥沢這種後期高火力，DP 要留到他 OP 峰值的後段。早期能擋就擋、擋不住就認賠保牌，**絕不把好防禦牌投進必敗判定**。本場 T6 認賠＝正解；對照上一場「太早打掉防禦牌→接不住連續高攻」＝反例。
- **出牌順序**：優先把「雙用途」卡放進它的位置——大見/小作→receive（棄牌成本即堆墓＋RCV 防守）、角名→attack（反打即推進目標）；北信介/宮侑等 Guts／6 種到位再下。讓每個動作同時推進 gameplan，而不是用純白板擋完就沒了。
