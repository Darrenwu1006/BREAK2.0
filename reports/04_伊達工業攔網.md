# 伊達工業「攔網軸」牌組 策略分析報告

> [Gemini 2026-06-19] 本報告基於 HV-P02 系列卡表撰寫，針對用戶常用對手（稲荷崎、青葉城西二彈改）進行對戰分析。

---

## 1. 牌組核心機制解析

伊達工業攔網軸是全遊戲中最具「防即攻」特性的牌組。其核心勝利引擎由三層循環構成：

### 第一層：多體攔網 → ドシャット 連鎖
本牌組的核心迴圈是**2-3名角色同時登場攔網**，觸發 ドシャット 效果，在成功攔截後將己方 OP 固定為 5 或 7，迫使對手下回合必須面對高強度守備挑戰，形成持續壓制。

| 效果來源 | 觸發條件 | ドシャット值 |
|---|---|---|
| 青根 高伸 (P02-037) | 攔網成功 | 4（側攔加 RCV-1 減弱） |
| 二口 堅治 (P02-038) | 攔網成功 | 4（側攔加 RCV-1 減弱） |
| 最強の防御で最速の攻撃（事件卡） | 攔網成功 + 對手 Event 區托球+攻擊≥4 | **7**（最高值） |
| バンチ・リード・ブロック（事件卡） | 攔網成功 + 攔網區≥2名伊達 | 5（≥3名時對手 RCV-1） |

**關鍵洞察**：ドシャット 的本質是「把防守轉化為下一輪的進攻壓力」。本牌組不以高 ATK 進攻，而是以「成功攔網後的反壓 OP」作為得分引擎，這讓 DP 計算比任何純攻擊型牌組都更加重要。

### 第二層：條件爆發型 BLK 值
本牌組有兩個 BLK 爆發角色，在條件滿足時可達 BLK8，遠超大多數攻擊型牌組的 OP：

- **鎌先 靖志 (P02-042)**：永續技能，對手 Event 區「托球卡 + 攻擊卡」合計 ≥ 4 時，BLK 從 3 **→ 8**。觸發後無需主動發動，攔網階段自動生效。
- **月島 螢 (HV-P02-003)**：攔網時主動觸發，對手 Event 區 ≥ 2 張時，丟手牌 Event 卡觸發 BLK+6（BLK **8**）。月島屬於烏野陣營，**跨牌組使用**，在伊達牌組中作為臨時爆發槍使用，但其觸發成本（需要持有 Event 卡）需要手牌管理配合。

> **月島跨牌組邏輯**：遊戲規則未禁止跨隊伍角色入牌組，但月島無法受益於任何「伊達工業角色」的 Buff 效果（如事件卡的 BLK+1 指定「伊達角色」），因此月島在本牌組中是**純攔截工具**，而非 Buff 承載體。使用時需注意事件卡的隊伍限制。

### 第三層：二口 堅治 039 的攻擊廢除技
**二口 堅治 (P02-039)** 的主動/攻擊效果是本牌組唯一的進攻型「控制技」：

> 「若二傳是伊達工業 S，下個對手回合，若對手攔網總 DP ≤ 6 則攔網失敗」

這個效果不是直接 ATK 提升，而是**讓對手的攔網行為變得無效**。DP ≤ 6 的攔網角色直接被「封鎖」，等同於己方進攻穿透所有低 BLK 的守備配置。搭配黃金川（伊達 S）二傳使用，是本牌組少數的主動進攻手段。

## 2. 遊戲計劃（Game Plan）

### 早期（Set 1-2）目標

**核心任務：建立攔網陣型，引導對手 Event 區積累**

1. **優先建立三席攔網配置**：
   - 中央攔網：鎌先（BLK3 基礎，等待觸發條件）或月島（依手牌 Event 卡存量決定）
   - 側攔：青根、二口038、笹谷、小原任選兩張
   - 目標是讓攔網區維持 2-3 名伊達角色，以便後期事件卡能觸發 ドシャット 5/7

2. **作並 浩輔的早期部署**：
   - 作並的觸發效果（手牌棄掉 → 牌庫頂3張全為伊達 → 免費側攔）在早期最有利用空間，因為牌庫尚未消耗，純伊達工業概率較高
   - 早期使用可以「以0成本補滿側攔位置」，節省手牌資源給事件卡

3. **不急於觸發 ドシャット**：
   - 早期 ドシャット4 的壓力相對有限，OP=4 對手能接球的概率仍高
   - 重點在於攔網成功積累，讓對手開始用 Event 卡回應

4. **控制 Set 消耗節奏**：
   - 伊達是守備型牌組，允許早期少量失分，只要不是被 ドシャット 高壓反殺即可
   - 盡量讓對手在 Set 1-2 就把攻擊型 Event 卡打進 Event 區，為鎌先/最強防御解鎖條件做準備

### 中期（Set 3-4）目標

**核心任務：觸發鎌先 BLK+5，發動 ドシャット 7 的死亡循環**

1. **監控對手 Event 區**：
   - 一旦對手 Event 區「托球卡 + 攻擊卡」合計 ≥ 4，鎌先 BLK 自動升至 8
   - 此時搭配「最強の防御」事件卡（攔網成功 → ドシャット 7），對手下回合將面對 OP=7 的壓力
   - 大多數接球角色 RCV 為 5-6，OP=7 需要 RCV ≥ 7 才能防守，形成高失分壓力

2. **多人攔網 + 事件卡搭配**：
   - 攔網區 ≥ 2 名伊達時使用「バンチ・リード・ブロック」觸發 ドシャット 5
   - 攔網區 ≥ 3 名伊達時，同卡額外效果：對手接球角色登場時 RCV-1，進一步削弱其防守值
   - 若同時有月島在場，「最強防御」的 BLK+1 因只針對「伊達角色」不會加到月島，需另計算總 DP

3. **追分 拓朗（監督 Event 卡）的靈活使用**：
   - BLK3 的伊達角色 BLK+1（讓鎌先/青根/二口039/笹谷/小原更穩）
   - 或選擇 RCV+1（強化防守面）
   - 中期是追分 Event 卡使用的黃金時段

### 後期（勝利條件觸發）

**核心任務：二口039 攔網失敗封鎖 + 連續 ドシャット 7 壓死對手**

1. **二口039 的攻擊控制**：
   - 確保黃金川在二傳位（TOS1 穩定），使用二口039 攻擊
   - 下個對手回合，DP ≤ 6 的攔網直接失敗
   - 此效果在後期對手攔網資源被消耗後效果最佳，因為對手難以補充 BLK≥7 的高攔網角色
   - **注意**：二口039 只有 BLK3，無 RCV，是進攻兼控制角色，需在不擔任攔網時發動效果

2. **連續 ドシャット 壓死戰術**：
   - 攻方 ドシャット 7 → 對手 Set 失守 → インターバル（對手從 Set 區拿1張補手牌）→ 再次攻守易位
   - 反覆循環，每輪迫使對手消耗 Set 卡，至對手 Set 區歸零宣告 Lost
   - 後期鎌先 BLK8 + 事件卡雙重防守，使對手很難突破，同時己方 ドシャット 7 持續輸出

## 3. 關鍵組合與連鎖

### 組合 A：鎌先 BLK8 + 最強防御 ドシャット 7（核心終結組合）

```
條件：對手 Event 區 托球+攻擊卡 合計 ≥ 4
配置：鎌先（BLK8）+ 任意一名側攔（BLK3）= 總 DP 11
事件：最強の防御 → 一名伊達 BLK+1（BLK9/4）= 總 DP 13
若攔網成功 → ドシャット 7
結果：對手下回合需 RCV ≥ 8 才能防守，幾乎無角色可達
```

### 組合 B：3人攔網 + バンチ・リード ドシャット 5 + RCV-1

```
條件：攔網區≥3名伊達工業角色
事件：バンチ・リード・ブロック → ドシャット 5 + 下個對手回合接球角色登場 RCV-1
效果：對手 RCV 被削弱後，下回合需要RCV≥6 的角色才能防OP=5
      若搭配青根/二口038的側攔效果，對手接球角色登場再-1 = RCV-2
結果：接球角色實際有效 RCV 被大幅削弱，OP=5 變得難以防守
```

### 組合 C：作並 浩輔 免費側攔 + 事件卡

```
觸發：棄掉作並手牌 → 牌庫頂3張全為伊達工業 → 從棄牌區拿1張無效果伊達角色以側攔登場
成本：0張手牌（作並本身即觸發）
用途：在攔網人手不足時緊急補位，或完成「3人攔網」觸發條件
優先棄掉作並的時機：攔網區空缺、牌庫仍有高濃度伊達卡時
```

### 組合 D：月島 螢 BLK8 + 事件卡爆發

```
條件：對手 Event 區 ≥ 2 張；月島使用者手中有Event卡
效果：月島主動發動 → 棄手牌Event卡 → BLK+6（BLK8）+ 抽1張
注意：「最強防御」的 BLK+1 僅限伊達角色，月島不受益
      因此月島在此組合中是 BLK8 的固定攔截機器，不疊加事件卡 Buff
搭配：中央鎌先+BLK8月島雙攔或月島獨攔，均能達到高 DP
```

### 組合 E：二口039 攻擊封印 + 黃金川二傳

```
條件：黃金川在二傳位（伊達 S），二口039 在攻擊位
效果：下個對手回合，攔網總 DP ≤ 6 的配置 → 攔網失敗
優勢：封鎖對手低 BLK 配置的攔網，使伊達的攻擊能直接得分
限制：二口039 需要在「不攔網」的回合發動，屬攻守分離型卡
```

### ドシャット 連鎖場景示意

```
回合 N：伊達攔網成功（DP≥OP）→ ドシャット7 觸發 → 己方OP=7
回合 N+1：對手需防守OP=7（接球方 RCV≥7 才能成功）
→ 若防守失敗：對手失分，Set 消耗
→ 若防守成功（RCV7+）：雙方回到平衡，但對手消耗了高規格接球角色
回合 N+2：伊達再次佈置攔網陣型，準備下輪 ドシャット
```

| 1 | 最強の防御 / バンチ・リード | 事件卡備用，攔網成功時觸發ドシャット |

**理想起手的判定標準**：
- 攔網位可以立刻排上 2 人（中央+側攔）
- 有二傳角色（黃金川）穩定二傳位
- 有事件卡準備在攔網成功時觸發ドシャット高值

### 可接受起手

- **缺事件卡型**：有攔網三人陣（鎌先+兩側攔）+ 黃金川，事件卡依靠後續抽取
- **缺黃金川型**：有完整攔網陣 + 事件卡，二傳用小原/笹谷暫代，但二口039 效果無法觸發
- **有作並型**：攔網人數不足時，作並可以補一席側攔，允許正式攔網角色少一張

### 必須換牌的情況（Mulligan）

以下情況建議重新抽取：

1. **全是接球/無BLK核心**：攔網值合計不足，早期必然失守
2. **無二傳角色且無月島**：二傳缺位讓攻擊效果全部無法運作
3. **6張全是事件卡或接近**：角色嚴重不足，無法建立攔網陣型
4. **無任何攔網角色（BLK值）**：無法建立攔網陣型，核心引擎失效
5. **只有月島，缺其他攔網角色**：月島條件觸發，缺少配合時難以建立穩定配置

## 5. 優勢分析

### ✅ 被動型勝利條件觸發
伊達攔網軸的勝利路徑是「對手越攻擊，觸發條件越容易滿足」。鎌先 BLK+5 和最強防御的 ドシャット 7 都依賴對手 Event 區積累，這意味著：
- 對手越積極打出攻擊/托球事件卡，伊達的防守就越強
- 對手若因怕觸發條件而保留 Event 卡，則其進攻效率下降

### ✅ 攔網成功即攻擊的「防守即得分」循環
ドシャット 後，伊達不需要額外的進攻牌就能形成 OP 壓力，這讓牌組在事件卡耗盡後仍能維持基本進攻能力。

### ✅ 多體攔網的高上限 DP
鎌先 BLK8 + 事件卡 BLK+1 + 側攔 BLK3 可以組出 DP 12+ 的守備值，能擋住大多數 ATK4-7 的高攻擊。

### ✅ 作並浩輔的0成本補位
免費側攔效果可以在關鍵回合補足人數，讓3人攔網條件更容易達成，且不消耗手牌。

### ✅ 二口039 的進攻控制具有唯一性
「攔網失敗」的控制效果在整個遊戲中是稀有機制，讓伊達的進攻具有其他牌組無法複製的突破能力。

### ✅ RCV-1 疊加的防守削弱連鎖
青根/二口038 的側攔效果（攔網成功後對手接球登場 RCV-1）+ バンチ・リード（3人攔網時）可以同時發動，對手接球角色每次登場都被削弱，逐漸失去防守能力。

## 6. 劣勢分析

### ❌ 二口039 需要對手「先攻擊一次」才能觸發
二口039 的攻擊效果是「下個對手回合，DP≤6 攔網失敗」，意味著：
- **己方必須先攻擊**才能讓效果生效於對手的防守
- 若己方是守備方（對手發球），二口039 效果完全無法觸發
- 效果只生效一回合，若對手臨時補充高 BLK 角色（BLK≥7），效果廢除
- 這讓伊達的進攻控制具有「延遲性」和「一次性」的天然限制

### ❌ 進攻資源匱乏
本牌組唯一的主動攻擊角色是二口039，其他角色幾乎都是攔網/接球型。對手若能持續突破ドシャット的 OP 壓力，伊達沒有高 ATK 的反擊手段。

### ❌ 月島是跨隊伍角色，受 Buff 限制
月島無法接受「一名伊達角色 BLK+1」的強化，在事件卡疊加 Buff 的場景下，月島始終保持固定 BLK8（觸發條件滿足時），無法進一步強化。

### ❌ 作並浩輔觸發不穩定
牌庫頂3張「全部是伊達工業」的概率在後期（牌庫變薄、已有多張伊達在場或棄牌區）會降低，中後期使用風險增加。

### ❌ 對非 Event 卡積累的對手，條件可能遲遲不達
鎌先、最強防御的條件都依賴「對手 Event 區托球+攻擊卡 ≥ 4」，若對手故意控制 Event 卡的使用，可能讓這兩個關鍵觸發條件長時間無法達到。

### ❌ 稲荷崎角名「無視中央攔網」的威脅（裁定更新）
角名倫太郎強化效果在棄牌6種後：攻擊時**無視對手中央攔網者的 BLK 值**。
裁定確認：**側邊攔網者 BLK 仍然完整計算**，僅中央位置的 BLK 被歸零。
因此鎌先/月島若擔任中央，其 BLK8 被廢除；但側攔的青根/二口038/笹谷/小原的 BLK3 照常生效。
這代表伊達面對角名的抵抗力**比原估計稍強**——只要能排出充足側攔，仍可形成有效防線。詳見第7節。

### ❌ 鎌先 BLK+5 為永續型，但本身 BLK3
未觸發時，鎌先的基礎 BLK3 對高 ATK 攻擊並無特別優勢，中央攔網若過度依賴鎌先，在條件未達成前可能漏球。

## 7. 對稻荷崎（堆墓角名）的應對策略

### 威脅等級評估：★★★★☆（高，但比原估計稍低——裁定修正後）

### 核心威脅：角名 倫太郎 的「無視中央攔網」（裁定①確認）

角名的強化效果（棄牌6種後，ATK+2，下個對手回合**無視對手中央攔網者的 BLK 值**）。

**裁定確認**：此效果**僅無視中央攔網角色的 BLK**，側邊攔網者 BLK 仍然完整計算。

**更新後影響計算**：

| 場景 | 中央 BLK | 側攔 BLK | 有效 DP | 角名 ATK | 結果 |
|---|---|---|---|---|---|
| 角名激活前（正常） | 鎌先 8 | 青根 3 | **11** | 4 | 防守成功 |
| 角名激活後（中央廢除） | ~~8~~ → 0 | 青根 3 | **3** | 4 | **防守失敗** |
| 角名激活後（2側攔） | ~~8~~ → 0 | BLK3 × 2 | **6** | 4 | **防守失敗**（DP=6，ATK4 = OP≥7時失敗） |
| 角名激活後（3側攔，無中央） | 無 | BLK3 × 3 | **9** | 4 | 防守成功（前提作並補位成功） |

**關鍵修正**：三人側攔配置（以作並浩輔免費補位為手段），在角名激活後仍能組出 DP9，而角名 ATK4 的 OP（托球TOS1 + ATK4 = OP5）需要 DP≥5 才能攔截，側攔 DP9 完全可擋。若宮治強化後 ATK=5（OP=6），三側攔 DP9 仍能防守。

### 雙向互動分析：角名 × 二口039 DP≤6 攔網失敗（裁定②確認）

**裁定確認**：二口039 效果「對手攔網總 DP ≤ 6 → 攔網失敗」，其 DP 計算包含**中央 + 所有側邊攔網者的 BLK 總值**。

這產生了一個關鍵的雙向互動：

**對伊達不利的方向（稲荷崎使用角名後，伊達防守）**：
```
角名效果激活 → 伊達中央（鎌先BLK8）被歸零
伊達側攔配置：
```

1. ドシャット 的「防守後反壓 OP」
2. 二口039 的「攻擊封印控制」

這讓本牌組在面對「高防守型」對手時（如 RCV7+ 的接球角色）會明顯吃力，但對「進攻型」對手（青葉城西、烏野）則有壓倒性優勢。
