# 音駒「二口干擾」牌組 策略分析報告

> **推薦推理等級：高**  
> 本牌組涉及磨庫干擾循環、跨回合狀態疊加（ATK-2 × 多次觸發）、Guts 資源管理，需精確計算觸發時序，不可輕率操作。

---

## 1. 牌組核心機制解析

### 1.1 雙重干擾軸（Disruption Core）

本牌組以**兩層干擾**疊加製造對手的資源危機，而非依靠單一爆發壓制：

| 干擾層 | 執行者 | 效果 |
|---|---|---|
| **磨庫干擾** | 二口 堅治 (HV-PR-027) | 對手下回合每次非抽牌補牌 → 棄牌庫頂3張 |
| **ATK 壓制** | 孤爪 研磨 (HV-D02-001) | 對手攻擊角色登場時 ATK-2（支付2Guts激活） |

**關鍵協同點**：兩種干擾均指向「對手的補牌行為」與「對手的出場行為」。對於依賴頻繁非抽牌補牌（如稻荷崎的北信介、青葉城西的及川PR事件）的對手，二口 PR 的磨庫效果可在同一回合觸發多次，形成指數級牌庫壓縮。

### 1.2 攔網反擊軸（Counter-Block Core）

- **黑尾 鐵朗 (HV-D02-002)**：主動支付1Guts，攔網成功時觸發 **ドシャット5**——本回合結束時自方 OP 升為5，直接換算成 Lost 壓力。BLK3 的高數值確保攔網成功率。
- **灰羽 列夫 (HV-D02-004)**：被動機動攔網——登場攻擊區時可支付2Guts 轉入攔網區，在 Guts 充裕時等同於額外多出一個攔網槽位。
- **犬岡 走 (HV-D02-008)**：BLK3/RCV3/ATK3 全能型，在防守端填補空缺、攻守俱佳。

### 1.3 攻擊軸（Offense Core）

本牌組的攻擊力屬**中上水準**，非爆發型：
- 灰羽 列夫 ATK3 + 孤爪 TOS1 搭配正常出力約為 OP=4～5。
- 犬岡走 ATK3、福永 ATK3、山本 猛虎 ATK3 提供穩定攻擊力。
- 黑尾 ドシャット5 的 OP=5 是最高潛在攻擊值，但依賴先完成攔網。

> **定位總結**：本牌組是「干擾中速」型——前期靠 ATK 削弱對手拿分、中期靠磨庫消耗對手資源、後期靠 ドシャット5 收尾。並非倚靠爆發一擊致命，而是讓對手的牌庫/手牌不斷惡化直至無法回應。

## 2. 遊戲計劃（Game Plan）

### 早期（Set 1–2）目標

1. **確立孤爪 研磨在舉球區**：儘快找到孤爪並用2Guts激活，讓對手的攻擊角色每次登場都承受 ATK-2 的懲罰。這是全牌組最重要的早期部署，必須優先完成。
2. **建立 Guts 節奏**：孤爪激活耗2Guts，灰羽機動攔網耗2Guts，黑尾 ドシャット 耗1Guts——三者同回合全部使用需要5Guts，早期要謹慎分配，不要讓 Guts 一次耗光。
3. **用 Event 卡穩定手牌**：`手は前っつったでしょう`（抽1張 + 補強黑尾或接球）和 `物理攻撃でブッ倒ればいいんだよね`（抽1張 + 篩牌庫）維持手牌品質，早期不要浪費。
4. **避免讓對手快速積累差距**：Set 1–2 若失分，盡量在 Interval 補滿手牌，為中期干擾蓄力。

**Set 1–2 理想狀態**：孤爪在舉球區發動 → 對手ATK普遍降至1或更低 → 己方靠 RCV 守穩，無需高DP。

### 中期（Set 3–4）目標

1. **二口 堅治 PR 進場啟動磨庫**：在對手開始積極補牌的回合（稻荷崎的棄牌回收、青葉城西的事件補牌）讓二口 PR 佔位，觸發下回合磨庫。每次對手非抽牌補1張 = 對手牌庫被磨3張，若對手單回合補3張 = 磨9張。
2. **孤爪 + 二口 雙干擾同時壓制**：孤爪限制攻擊力、二口消耗牌庫。對手在這個階段通常面臨「進攻也沒用、補牌會被磨」的雙重困境。
3. **黑尾攔網創造 ドシャット 機會**：當對手 ATK 被孤爪削弱至2以下，攔網成功率大幅提升，此時黑尾 ドシャット5 創造的 OP=5 足以取下關鍵分。
4. **灰羽列夫機動攔網輔助**：靈活應對對手的換人/換位策略，不讓攔網槽位被對手規避。

### 後期（勝利條件觸發）

1. **牌庫壓縮勝利（磨庫路線）**：二口 PR 在關鍵回合反覆觸發，若對手牌庫剩餘不多時，被磨空或無法 Draw 即可強制 Lost。
2. **ドシャット5 收尾（攔網反擊路線）**：在對手 ATK 被孤爪壓到無法得分的情況下，黑尾的 OP=5 主動進攻奪取最後幾個 Set。
3. **Set 流失控制**：若本方 Set 卡較少時，優先靠穩固接球（夜久 RCV7 或海 RCV5）守住，等待干擾效果結束對手資源。

> **注意**：本牌組沒有快攻爆發手段，後期拉鋸時若 Guts 不足，應選擇保留黑尾技能，優先靠 ATK3 角色穩定得分，而非盲目啟動 ドシャット。

## 3. 關鍵組合與連鎖

### 3.1 二口 PR 磨庫的最佳觸發時機

**理想觸發條件**：
- 對手預計在下個回合使用「非抽牌補牌」效果（如北信介、及川PR事件、棄牌區回收等）
- 二口 PR 在本回合**發球、攔網或攻擊**任一階段在場即可激活

**操作流程**：
```
本回合：二口 PR 以任意方式在場（發球角色/攔網角色/攻擊角色）
       ↓
效果觸發：下個對手回合，對手每次非抽牌補牌 → 棄對手牌庫頂3張
       ↓
對手回合：若對手補3張非抽牌手牌 → 其牌庫共被磨9張
```

**最大磨庫場景**（對稻荷崎）：
- 對手用北信介（補1張）+ どや回收（補3張）= 4次補牌 → 磨12張
- 配合孤爪ATK-2使對手進攻無效，對手只能繼續補牌應急 → 磨庫更深

**注意事項**：
- 二口 PR 的效果是「**下個**對手回合」，所以需要提前1回合佈局
- 若對手這回合沒有非抽牌補牌，效果空轉，需謹慎選擇觸發時機

### 3.2 黑尾 ドシャット5 的觸發條件與決策

**觸發前提**：黑尾鐵朗作為攔網角色登場，支付1Guts，**攔網成功（BLK ≥ OP）** 時觸發。

**攔網成功率估算**：

| 對手有效ATK（孤爪削弱後） | 黑尾BLK3成功率 |
|---|---|
| ATK=1 | 幾乎必成功（OP=TOS+1） |
| ATK=2 | 高概率（OP=TOS+2，通常≤3） |
| ATK=3 | 中等（若TOS=1, OP=4，需搭配其他BLK） |

**ドシャット5 後的 OP 時序**：
- 攔網成功（黑尾技能觸發）→ 本回合結束時己方 OP 變為5
- 此 OP=5 用於得分，非對手攻擊反制，注意時序確認

**最佳使用場景**：已知對手下個防守角色 RCV ≤ 4 時，ドシャット5 保證得分。

### 3.3 灰羽列夫機動攔網的 Guts 管理

**機動攔網邏輯**：
```
灰羽列夫登場到攻擊區（正常登場）
       ↓
若判斷本回合對手攻擊 OP 較高，支付2Guts
       ↓
灰羽列夫轉入攔網區，補強 BLK（原BLK2）
```

**Guts 分配優先序**：
1. 孤爪激活（2Guts）— **最優先**，每回合持續壓制
2. 黑尾ドシャット（1Guts）— 攔網成功後加乘
3. 灰羽機動（2Guts）— 情況允許時使用
4. 夜久棄牌+RCV（免費棄手牌）— 不耗Guts，優先考慮

> **5 Guts 組合技**：孤爪(2) + 灰羽機動(2) + 黑尾ドシャット(1) = 5Guts 全開，理論上可同時壓制ATK+補強BLK+ドシャット反擊，但需要 Guts 積累足夠才能執行。

## 4. 起手最佳化（Opening Hand Strategy）

### 最理想起手（6張）

| 構成 | 理由 |
|---|---|
| **孤爪 研磨 × 1–2** | 核心干擾源，越早上場越好 |
| **黑尾 鐵朗 × 1** | 攔網 + ドシャット核心 |
| **二口 堅治 PR × 1** | 磨庫干擾，中期出場 |
| **Event 卡 × 1–2**（手は前 or 物理攻撃） | 穩定手牌品質，確保後續能找到關鍵牌 |
| **接球角色 × 1**（海/夜久/福永） | 早期防守穩定 |

**理想開局範例**：
- 孤爪 × 1 + 黑尾 × 1 + 二口PR × 1 + 手は前 × 1 + 物理攻撃 × 1 + 犬岡走 × 1

### 可接受起手

- 缺少孤爪但有2張 Event 卡 + 黑尾 → 靠 Event 篩牌找孤爪
- 缺少二口PR但有強力攔網陣容（黑尾+灰羽+犬岡） → 先打穩攔網，等二口PR入手
- 手牌全為接球/攻擊角色 + 1張 Event → 靠Event補充手牌並找核心牌

### 必須換牌的情況（Mulligan）

以下情況建議全換或至少換3張以上：

- **完全沒有孤爪、黑尾、二口PR**：三大核心全缺，手牌方向錯誤
- **6張全是攻擊角色，無Event無核心**：缺乏手牌加速，早期被快速壓制
- **3張以上相同名稱角色（重複過多）**：角色區只能放1個名稱，重複牌佔手牌空間但無法出場
- **手牌全為高費技能角色，Guts 來源不穩定**：開局就無法啟動任何技能

## 5. 優勢分析

### ✅ 穩定的多回合干擾壓力

孤爪研磨的 ATK-2 效果是**持續型**（下個對手回合每次攻擊角色登場時觸發），而非一次性。對於每回合都需要出場攻擊角色的對手（幾乎所有牌組），此效果等同於長期存在的 ATK 稅。對手若要維持ATK=3，在孤爪存在的情況下需要 TOS 達到4才能出力OP=4（TOS3 + ATK3-2 = OP=4才夠），大幅提高對手的維持成本。

### ✅ 磨庫干擾的滾雪球效應

二口 PR 的磨庫在對抗需要大量非抽牌補牌的牌組時呈滾雪球狀——對手補越多、牌庫越薄，越接近無法 Draw 的狀態。這對於依賴棄牌區回收的牌組（稻荷崎）尤其致命：被磨的牌不只是消耗，也永遠失去了回收機會。

### ✅ 靈活的攔網配置

BLK3 的黑尾 + 可機動轉位的灰羽列夫 + BLK3 的犬岡，使攔網陣容非常靈活。對手難以通過選擇特定位置規避攔網。

### ✅ 夜久/海 提供的接球韌性

夜久 衛輔 RCV7（棄1手牌）、海 信行 RCV5（無技能），提供穩定的防守底線。即使對手衝破干擾，接球成功率維持在合理水準。

### ✅ 事件卡效率高

8張 Event（上限）且分工明確：`手は前` 防守期抽牌+補強；`物理攻撃` 攻擊期篩牌+找關鍵角色。Card advantage 穩定。

## 6. 劣勢分析

### ❌ 依賴對手行為觸發干擾

二口 PR 的磨庫需要「對手以非抽牌方式補牌」才能生效。若對手本回合純靠抽牌維持手牌（例如靠 Draw Phase 正常抽），二口 PR 完全空轉。部分牌組設計可以規避「非抽牌補牌」，直接削弱磨庫軸的效能。

### ❌ Guts 資源高度緊張

三大技能（孤爪2G + 灰羽2G + 黑尾1G）若要同回合全開，需要5Guts。而 Guts 補充速度有限，若 Interval 補充不及，某一回合將被迫放棄部分技能。Guts 管理失誤是本牌組最常見的操作錯誤。

### ❌ 攻擊力上限偏低

本牌組最高 ATK 為3（灰羽、犬岡、福永、山本），搭配 TOS1 孤爪最高 OP=4（若無特殊加成）。對於高接球對手（RCV6+），得分困難。ドシャット5 雖然是突破口，但需要先完成攔網才能觸發，並非穩定輸出手段。

### ❌ 缺乏快速勝利路線

本牌組是慢性消耗型，沒有一擊致命的爆發手段。若對手能在前期（Set 1–2）快速積累大量 Set 差距，後期磨庫很難反追。面對快攻牌組尤為明顯。

### ❌ 芝山 優生 × 1 的邊緣角色

牌組中唯一1張的芝山/手白，作用有限，主要是為了湊40張或填補特定情況，實際上手率低且效果不穩定。

## 7. 對稻荷崎（堆墓角名）的應對策略

### 稻荷崎核心弱點分析

稻荷崎牌組的運作核心是：
1. 積累棄牌區6種不同稻荷崎角色
2. 靠北信介、どや俺の仲間、尾白阿蘭等持續從棄牌區/非抽牌方式補牌
3. 棄牌6種後啟動宮治ATK5、角名無視中央BLK等強化效果

**所有的補牌行為（北信介、どや、尾白）幾乎都是非抽牌方式** → 這正是二口 PR 磨庫的最佳目標。

### 7.1 二口 PR 壓制稻荷崎的非抽牌補牌

| 稻荷崎非抽牌補牌來源 | 每次觸發磨庫 |
|---|---|
| 北信介（棄手牌+3Guts → RCV+1，從Event區補1張） | +3張被磨 |
| どや俺の仲間（從棄牌區補最多3張） | +9張被磨（最多3次×3）|
| 尾白 阿蘭（2Guts → 從棄牌補1張稻荷崎） | +3張被磨 |
| 宮 侑HV-P02-017（3Guts → 從棄牌補1張WS/MB） | +3張被磨 |

**磨庫計算範例**：對手一回合使用北信介 + どや補3張 = 4次非抽牌補牌 → **磨12張**。稻荷崎牌庫40張，若早期就開始磨，中期可能只剩15–20張，後期極度空虛。

### 7.2 孤爪 ATK-2 對宮治/角名的壓制

| 稻荷崎攻擊者 | 原始 OP | 孤爪削弱後 OP（TOS=1基準） |
|---|---|---|
| 宮 治（ATK3 正常） | TOS+3 = 4 | TOS+3-2 = 2（幾乎無效攻擊）|
| 宮 治（ATK5 強化後） | TOS+5 = 6 | TOS+5-2 = 4 |
| 角名（ATK2 + 棄牌6種+2 = ATK4） | TOS+4 = 5 | TOS+4-2 = 3 |

> **孤爪對宮治強化版（ATK5）的削弱仍是有效的**——OP從6降至4，黑尾BLK3已無法單獨攔截，但配合犬岡BLK3可能組成 BLK6，勉強應對。

### 7.3 角名「無視中央攔網」的應對

角名 倫太郎 效果：棄牌6種時斜放發動，ATK+2，**下個對手回合無視中央攔網值**。

**應對策略**：
1. **優先阻止棄牌達到6種**：二口PR磨庫是磨**牌庫頂**而非棄牌區，無法直接消除對手已積累的棄牌。所以需要在稻荷崎完成6種之前就開始磨庫，讓其關鍵角色（宮侑/宮治/角名）在牌庫被磨掉，無法進入棄牌區積累。
2. **在角名技能觸發前讓稻荷崎牌庫接近枯竭**：若對手牌庫被磨到只剩5張，就算角名無視中央攔網，對手也很快無法 Draw。
3. **使用側邊攔網輔助**：灰羽列夫機動轉入攔網區，提供額外 BLK2，搭配犬岡 BLK3 = 總BLK5，配合孤爪削弱後的角名 OP=3，勉強可以應對。
4. **接球為主**：角名無視中央攔網後，若攔網失敗，靠夜久 RCV7 或海 RCV5 接球保底，不要賭攔網。

### 7.4 整體稻荷崎對局節奏

```
Set 1–2：建立孤爪，阻止稻荷崎早期積累ATK優勢
Set 2–3：二口PR進場，開始壓制北信介/どや的補牌循環
Set 3–4：稻荷崎牌庫快速薄化，對手難以維持6種棄牌數量所需的重新補充
後期：稻荷崎若牌庫接近空，被迫用弱牌防守，ドシャット5收尾
```

**評估**：**有利對局（音駒略有優勢）**。稻荷崎的補牌循環幾乎全是非抽牌方式，是二口PR最理想的目標，且棄牌機制仰賴特定牌在棄牌區，磨庫會讓這些牌永遠消失。但若稻荷崎能在第2回合前快速達到棄牌6種，角名+宮治的爆發需要承受。

## 8. 對青葉城西（二彈改）的應對策略

### 青葉城西核心弱點分析

青葉城西的核心是及川 徹 的連鎖——發球+TOS+棄牌手牌補強，並搭配「俺も思った☆」事件強化干擾（對手非抽牌補牌時棄1）。

**關鍵問題：及川「俺も思った☆」與二口 PR 是否形成干擾循環？**

### 8.1 干擾循環分析

**俺も思った☆ 效果**（若二傳/發球是及川）：下個對手回合，對手非抽牌方式獲得手牌時，對手棄1張。

**二口 PR 效果**：下個對手回合，對手非抽牌補牌時，棄對手牌庫頂3張。

**重要裁判釐清**：
- 「俺も思った☆」讓**音駒方**（從青葉城西視角看是對手）每次非抽牌補牌時**棄手牌**
