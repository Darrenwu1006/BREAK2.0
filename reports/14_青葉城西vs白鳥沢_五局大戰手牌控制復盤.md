## 0 規則前提

1. **防守登場限制**：
   - 「接球（Receive）」只能登場 **1 張** 角色卡，DP 即為該卡的 Receive 點數（加計技能修正）。
   - 「攔網（Block）」可登場 **1~3 張**（1 張必為 Center，最多 2 張 Side），且同名卡不能在攔網區同時存在兩張以上。
   - 參數為「－」的角色不能登場到對應區域（例如 `渡 親治` 攔網為「－」，無法用於攔網）。

2. **防守點數天花板**：
   - 青葉城西（二彈改）的接球手為 `渡 親治`[－/－/5/1/0]✦ 與 `国見 英`[1/0/5/0/0]✦。
   - 渡親治發動登場技能（棄 1 張手牌使接球 +1 并抽 1）可將單體接球 DP 推至最大值 **6**。
   - 攔網端以 `岩泉 一`[1/3/1/0/2]✦ 為主，單卡攔網 DP 最大為 **3**。若無其他具備 Block 點數的角色協防，攔網 DP 極低。

## 1 核心結論

1. **失 Set 歸因為機制死局**：
   - Set 3（Step 92，對手 OP 7 vs 我方 DP 6）與 Set 4（Step 127，對手 OP 8 vs 我方 DP 6）的失敗並非當下防守選擇錯誤，而是我方防守點數天花板（6 點）在機制上無法防住對手疊加事件與 Guts 爆發出來的超高 OP。
   - 一旦讓白鳥沢順利鋪設場地（打出多張 `クソ かっけえええ…!!!` 與 `鷲匠 鍛治`），並由 `牛島 若利` 疊加登場，OP 7+ 對青葉城西是不可防禦的。

2. **勝負手在於手牌控制（Hand Control）**：
   - 白鳥沢是高爆發、不依賴「非抽卡回收手牌」的牌組。這導致我方的 `俺も思った☆`（非抽卡補牌懲罰）在中前期形同虛設，無法壓制對手。
   - 決勝局（Set 5）玩家之所以獲勝，是因為成功透過事件卡 `お前達は 強いよ` 進行強力的手牌剝奪，使 AI 在最後關頭（Step 172）手牌僅剩無法登場於接球區的角色與事件卡，因「無法登場接球角色」而直接宣告 Lost。

## 3 為什麼（卡值/資源/對手牆）

1. **白鳥沢的資源爆發鏈**：
   - 白鳥沢通過早期的 `天童 覚` 與 `山形 隼人` 積攢了大量 Guts 點數，且在事件區打出了多張 `クソ かっけえええ…!!!` 與 `鷲匠 鍛治`（永久增加白鳥沢角色的進攻引導）。
   - `牛島 若利` 具備「疊加登場」的特性，可在手牌低於 3 張時獲得額外 +2 AP。這與白鳥沢消耗手牌鋪場的節奏完美契合。
   - 當這些條件在 Turn 3/4 飽和時，白鳥沢一拳能打出 7-8 點 OP，完全超出了青葉城西的單卡防禦極限。

2. **`俺も思った☆` 的無效化**：
   - 青葉城西（二彈改）依賴 `俺も思った☆` 來懲罰對手的檢索與回收，進而壓制手牌。
   - 然而白鳥沢不使用檢索，而是直接利用 Guts 區資源與牛島的 Guts 登場技能。這使得 `俺も思った☆` 毫無用武之地，白鳥沢的手牌在進攻前得以保持在安全線，免受棄牌懲罰。

### 戰術建議：
面對白鳥沢這類「防不住」的高爆發牌組，不能採取防守拖延戰術。應當在進攻回合：
1. **速攻優先**：盡快打出 `お前達は 強いよ` 進行手牌壓制。
2. **虛攻時機**：一旦對手手牌被消耗到 3 張以下，立刻登場 `国見 英` 進行 `虛攻(4)` 封鎖攔網，在對手爆發前拿下 Set。
