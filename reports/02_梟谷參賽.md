## 1. 牌組核心機制解析

梟谷「參賽」牌組的設計哲學是**穩定輸出 + 被動奪點**：藉由高TOS的赤葦京治完成強力舉球，交由木兎光太郎以巨高ATK完成攻擊，再以事件卡的永久削減效果壓制對手，最終讓對手無法有效攔截。

### 核心引擎結構

| 角色 / 卡牌 | 職能 | 關鍵數值 |
|---|---|---|
| 赤葦 京治 (P01-046) | 主要二傳 | S/RCV3/TOS2，無成本穩定 |
| 赤葦 京治 (P02-066) | 備用/爆發二傳 | TOS1→棄牌庫頂3張+2Guts→TOS+2（實質TOS3） |
| 木兎 光太郎 (P01-044) | 核心攻擊手 | WS/S5/ATK3，6張高密度 |
| 鷲尾 辰生 (P01-051) | 反制守門員 | MB/BLK3，ワンタッチ3條件觸發 |
| 全員があなたを見ていますよ | 永久場效控制 | 中央攔網者每次登場BLK-2 |
| 音駒じゃねえぞ！ | 攻守兩用 | 接球RCV+1 / 棄攔網角色→對手ATK OP-1 |

### 主要勝利引擎

**路線A — 赤葦TOS→木兎ATK爆發**
- 最優解：赤葦P01-046（TOS2）+ 木兎P01-044（ATK3）= OP值 **TOS2 + ATK3 = 5**
- 若以赤葦P02-066爆發：TOS3 + ATK3 = OP **6**
- 若木葉秋紀疊加（ATK+1）：OP可達 **TOS2 + ATK4 = 6 / TOS3 + ATK4 = 7**（上限極高）

**路線B — 全員があなたを見ていますよ永久削弱**
- 成立條件：二傳為赤葦、攻擊為木兎，事件卡使用後**下個對手回合開始永久生效**
- 效果：對手中央攔網角色「每次登場時BLK-2」
- 實戰意義：對手的MB角色即使BLK3也實質變成BLK1，有效瓦解對手的大BLK防線

**路線C — 鷲尾辰生ワンタッチ反制**
- 條件：對手OP≥4時，棄牌庫頂1張，若是梟谷角色則觸發「ワンタッチ3」
- ワンタッチ(3)：對手ATK的OP -3，並立即跳過攔網階段直接進入己方抽牌
- 實戰意義：高效截斷對手強攻，同時保留己方攔網機會

## 2. 遊戲計劃（Game Plan）

### 早期（Set 1-2）目標

**優先事項：建立手牌密度 + 確認二傳體系**

1. **確立赤葦為二傳**：盡可能在第一回合使用赤葦P01-046（穩定TOS2無成本）作為二傳區角色，建立穩定的進攻基盤。
2. **堆疊Guts**：早期優先選擇小見春樹（RCV6無技能）作為接球主力，Guts消耗較少的回合讓Guts庫存增長。
3. **保留事件卡時機**：「全員があなたを見ていますよ」不急著打出，等到赤葦+木兎組合確認後再觸發。
4. **手牌管理**：梟谷的事件卡數量恰好在上限8張，不浪費早期的抽牌資源。

**目標：** 穩定度過Set 1-2，不在早期Set浪費過多資源，讓Guts蓄積以備P02-066爆發。

---

### 中期（Set 3-4）目標

**優先事項：啟動全員があなたを見ていますよ + 建立攔網牆**

1. **觸發全員があなたを見ていますよ**：在二傳赤葦、攻擊木兎的回合打出此事件卡，從此對手每次嘗試用MB攔網都面臨BLK-2的持續壓力。
2. **鷲尾辰生登場**：把鷲尾放入攔網區，對對手OP≥4的強攻進行ワンタッチ反制，破壞對手進攻節奏。
3. **赤葦P02-066擇機爆發**：當Guts充足時（需2Guts + 棄牌庫頂3張），TOS可達3，配合木兎ATK3 = OP6，一舉拿下高分回合。
4. **音駒じゃねえぞ！的使用時機**：對手發動強攻前，在接球階段棄一名攔網區角色，使對手ATK OP-1，有效削減對手得分。

---

### 後期（勝利條件觸發）

**核心：利用已建立的BLK削弱場效逼對手Lost**

1. **全員があなたを見ていますよ生效後**，對手的中央攔網角色登場時BLK-2，梟谷的OP6-7幾乎可以穩定越過對手防線，讓對手持續流失Set卡。
2. **維持木兎在攻擊位**：木兎P01-044共6張，補充不難，確保攻擊端穩定。
3. **猿杙大和P01-049**（BLK3/ATK3）可在攻守轉換時登場，作為攔網兼攻擊的彈性使用。
4. **最終局**：若對手Set卡剩1-2張，優先選擇保守接球策略積累Guts，再以爆發回合完成致命一擊。

## 3. 關鍵組合與連鎖

### 組合一：「天才二傳 × 王牌攻擊手」爆發

```
赤葦P01-046（TOS2）× 木兎P01-044（ATK3）= OP5（基礎線）
   ↓ 追加
木葉秋紀被動（ATK+1）→ OP6
赤葦P02-066爆發（TOS3）→ OP7（木葉輔助下）
```

- **消耗**：P02-066需要2Guts + 棄牌庫頂3張（有不確定性風險）
- **風險管理**：棄牌庫頂3張會觸發隨機性，但梟谷密度高（木兎6張），整體穩定

---

### 組合二：「全員があなたを見ていますよ」永久場效

```
條件：二傳赤葦 + 攻擊木兎 → 打出「全員があなたを見ていますよ」
效果：下個對手回合起，對手中央攔網者每次登場BLK-2（永久）
連鎖：即使對手換攔網角色，每次重新登場仍會BLK-2
```

- **觸發成本**：抽1張 + 木兎ATK+1（本回合加值）
- **評價**：這是梟谷最強的場效控制，一旦生效，對手MB線幾乎癱瘓

---

### 組合三：「鷲尾辰生」ワンタッチ截斷

```
前提：鷲尾在攔網區 + 對手OP≥4
動作：棄牌庫頂1張，若是梟谷角色 → ワンタッチ(3)觸發
效果：對手ATK OP -3，立即跳過攔網階段，進入己方抽牌
```

- **命中率**：牌組內梟谷角色佔大多數，棄頂命中率高（粗估約75-80%）
- **評價**：有效反制對手的OP5-6高攻，甚至可能讓原本失分轉為守住

---

### 組合四：「音駒じゃねえぞ！」雙模式使用

```
模式A（接球）：抽1張 + 一名梟谷角色RCV+1 → 強化接球防守
模式B（攔網削弱）：棄一名攔網區梟谷角色 → 對手ATK OP-1
   ※ 模式B配合鷲尾ワンタッチ：對手原本OP6 → ワンタッチ-3 → 音駒-1 = 剩下OP2，幾乎必守
```

---

### 組合五：木葉秋紀「上方有梟谷角色」被動

```
條件：攻擊角色上方有梟谷角色登場 + 棄1張手牌
效果：攻擊角色ATK+1
```

- **使用時機**：Guts充足時，手牌有冗餘時搭配使用
- **評價**：手牌消耗換取ATK+1，邊際效益依手牌狀況而定，不宜強求

## 5. 優勢分析

- 赤葦P01-046（5張）+ P02-066（5張）= 共10張二傳角色，命中率極高
- TOS2穩定基礎，TOS3爆發上限，攻擊端輸出一致性強

### ✅ 攻擊密度高
- 木兎P01-044共6張，ATK3為高密度主力，補牌容易，不怕消耗
- 搭配木葉秋紀ATK+1可進一步突破防線

### ✅ 場效控制唯一性
- 「全員があなたを見ていますよ」的中央攔網BLK-2永久場效在現有牌池中極具稀缺性
- 一旦觸發，對手改變攔網策略的成本極高（換角色也無效）

### ✅ 接球防守厚實
- 小見春樹（RCV6）×7 + 赤葦P01-046（RCV3）×5，接球值偏高
- 猿杙大和P-026（主動RCV+4）在Guts充足時可達RCV7，防守上限驚人

### ✅ ワンタッチ反制機制
- 鷲尾辰生提供條件式的「主動截斷」，讓對手難以放心發動高OP攻擊

### ✅ 事件卡剛好滿8張
- 恰好達到上限，手牌質量密度最大化，每張事件卡都有確切用途

## 6. 劣勢分析

### ❌ 赤葦P02-066的不確定性風險
- 棄牌庫頂3張的動作有隨機性，可能把關鍵角色（木兎/赤葦）送入棄牌區
- 棄牌區在本牌組沒有回收機制，損失不可逆

### ❌ BLK端薄弱
- 鷲尾辰生（BLK3）×3 + 猿杙P01-049（BLK3）×1 = 僅4張攔網角色
- 若對手攻擊OP不達≥4，鷲尾ワンタッチ無法觸發，攔網能力驟降
- 對手選擇「穩定中OP」進攻策略（OP3-4）時，攔網壓力較大

### ❌ 無棄牌區回收機制
- 牌組沒有任何從棄牌區撿回角色的效果
- 長局戰不利，對手若能拖延Set進程，梟谷的牌庫深度優勢會逐漸消耗

### ❌ 事件卡的觸發條件嚴格
- 「全員があなたを見ていますよ」需要同時有赤葦二傳+木兎攻擊，才能觸發永久場效
- 若對手能破壞此組合（如手牌干擾），事件卡就只是抽1張+ATK+1的普通牌

### ❌ Guts管理壓力
- 猿杙P-026 RCV+4需要消耗4Guts，P02-066需要2Guts + 棄牌庫頂
- Guts管理不佳時，後期爆發能力受限

## 7. 對稻荷崎（堆墓角名）的應對策略

### 威脅評估

稻荷崎的核心威脅來自於**棄牌區6種解鎖**後的爆發：
- 宮治P-048：ATK5（2Guts解鎖），爆發OP = TOS + 5，非常高
- 角名倫太郎：斜放ATK+2後「下個對手回合無視中央攔網值」，直接繞過梟谷的BLK牆

### 鷲尾ワンタッチ能否有效干擾？

```
稻荷崎最高OP = 宮侑TOS（最高TOS3）+ 宮治ATK5 = OP8
→ 明顯≥4，鷲尾ワンタッチ觸發條件滿足

觸發效果：OP -3 → 稻荷崎實際OP = 5
→ 梟谷RCV端：小見RCV6 / 猿杙主動RCV7 ← 仍可守住！
```

**但！** 角名解鎖「無視中央攔網」時，需確認是否覆蓋整個攔網階段技能的發動。若角名效果覆蓋鷲尾的ワンタッチ技能，則鷲尾可能無法發動。

> **建議**：在角名解鎖前，優先消耗稻荷崎的Guts（用OP壓制），讓對手難以達成6種解鎖條件。

### 音駒じゃねえぞ！的干擾

- 在對手攻擊OP已形成（宮治ATK5）時，於接球前棄攔網角色 → 對手ATK OP-1 = OP7
- 搭配鷲尾ワンタッチ(3) → 稻荷崎最終OP = 4
- 梟谷小見RCV6 / 猿杙RCV7，**有機會守住宮治的全力攻擊**

### 全員があなたを見ていますよ的反制價值

- 稻荷崎沒有強力的MB攔網體系，此場效對稻荷崎的壓制效果**較有限**
- 但只要梟谷的OP能夠穩定在5-6，稻荷崎的接球（赤木RCV6、尾白RCV3+3）仍面臨壓力

### 應對重點

| 策略 | 說明 |
|---|---|
| 優先壓制Guts | 早期以高OP攻擊，逼迫稻荷崎消耗接球Guts，延遲6種解鎖 |
| 鷲尾提前上場 | 在稻荷崎解鎖前，將鷲尾預置攔網區，隨時準備ワンタッチ |
| 全員あなた早觸發 | 爭取在Set 2-3前觸發場效，即使對稻荷崎效果有限，至少ATK+1輔助進攻 |
| 警戒角名無視攔網 | 角名登場後，攔網阻力幾乎歸零，此時應專注接球端（猿杙主動RCV+4）守住 |
| 手牌管理優先 | 稻荷崎的北信介/大見太郎有抽牌循環能力，手牌差距可能擴大，需謹慎棄牌 |

## 8. 對青葉城西（二彈改）的應對策略

### 威脅評估

青葉城西的核心干擾來自**及川徹的手牌壓制**：
- 及川P01-033：棄1張青葉城西手牌後，「若對手手牌≥4，對手棄1張」
- 俺も思った☆（EVENT）：下個對手回合「非抽牌方式獲得手牌時棄1張」
- 持續手牌干擾下，梟谷的事件卡觸發條件和赤葦P02-066的Guts消耗都會受到壓縮

### 木兎高ATK vs 青葉城西高接球

```
青葉城西接球：
- 渡親治 RCV5 ×6（無技能穩定）
- 国見英 RCV5 ×4（穩定接球）
- 国見英P02-058（2Guts：手牌抽至3張）

梟谷攻擊上限：
- 赤葦TOS2 + 木兎ATK3 = OP5（基礎）
- 赤葦P02-066爆發 + 木葉輔助 = OP7（上限）
```

**分析**：青葉城西RCV5是標準防線，梟谷OP5基礎線只能持平。**必須啟動爆發組合（OP6-7）才能穩定突破。**

### 如何維持手牌質量

| 問題 | 解決方案 |
|---|---|
| 及川手牌干擾（對手手牌≥4→棄1） | 在及川發動技能前，主動將手牌維持在3張以內（低於4張門檻） |
| 俺も思った☆（非抽牌獲手牌棄1） | 避免在對手下一回合透過非抽牌方式（如事件卡效果）獲得手牌；優先使用抽牌效果補牌 |
| 手牌被壓至2-3張時 | 使用闇路建行（查牌庫頂3張，最多1張梟谷加手牌）精準補充關鍵角色 |
| 手牌不足導致木葉被動無法發動 | 木葉被動「棄1張手牌→ATK+1」在手牌≤3時應慎用，不要強行消耗 |

### 全員があなたを見ていますよ的關鍵價值

- 青葉城西的攻擊核心是岩泉一（ATK5），攔網端僅及川P02-056（BLK2）
- 觸發全員あなた後，及川P02-056的BLK2 → 實質BLK0（登場即-2）
- 這意味著**青葉城西幾乎失去攔網能力**，梟谷OP5基礎線已足以突破

### 應對及川岩泉高攻

```
岩泉P01-035：ATK5（WS，無技能）→ 及川TOS + 岩泉ATK5 = OP最高6-7
→ 梟谷接球：小見RCV6 / 猿杙主動RCV7 ← 可守住OP6，壓力極大
→ 鷲尾ワンタッチ(3)：OP7 → 4，梟谷RCV6即可守住

岩泉P02-057（ATK3）：接球：對手手牌≤3時3Guts→RCV+7（總RCV10！）
→ 此時梟谷OP7也可能被守住
→ 建議：避免手牌降至≤3，維持手牌≥4以阻斷岩泉P02-057的接球技能觸發
```
